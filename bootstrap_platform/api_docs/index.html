API Documentation

Base URL: https://api.commission-intelligence.io/v1

The Commission Intelligence Platform API is a RESTful API that provides programmatic access to all platform features. Build custom integrations, automate workflows, and access your commission data from any application. 

# Quick test with curl
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.commission-intelligence.io/v1/policies

Authentication

The Commission IQ API uses Bearer token authentication. Include your API key in the Authorization header of each request.

Getting Your API Key

  1. Log in to your Commission IQ dashboard
  2. Navigate to Settings â†’ API Keys
  3. Click "Create New Key" and give it a descriptive name
  4. Copy the key immediately - it won't be shown again

Request Example

curl -X GET \
  https://api.commission-intelligence.io/v1/policies \
  -H "Authorization: Bearer cipk_live_abc123..."

Using with SDKs

import commission_iq

client = commission_iq.Client(
    api_key="cipk_live_abc123..."
)

Rate Limits

To ensure fair usage and system stability, the API enforces rate limits based on your plan.

Plan Requests/Second Requests/Day Burst Limit
Starter 10 10,000 20
Professional 50 100,000 100
Enterprise 100 Unlimited 200

Rate Limit Headers: Every response includes headers showing your current usage:

  • X-RateLimit-Limit - Your rate limit
  • X-RateLimit-Remaining - Requests remaining
  • X-RateLimit-Reset - Reset timestamp

Policies

Manage insurance policies and their associated commission data.

List Policies

GET

Retrieve a paginated list of policies with optional filtering.

GET /policies

Query Parameters

Parameter Type Description
page integer Page number (default: 1)
per_page integer Results per page (max: 100)
status string Filter by status
carrier string Filter by carrier name

Example Request

curl -X GET \
  "https://api.commission-intelligence.io/v1/policies?page=1&status=active" \
  -H "Authorization: Bearer YOUR_API_KEY"

Example Response

{
  "data": [
    {
      "id": "pol-1234",
      "policy_number": "AUTO-123456",
      "customer": "ABC Insurance",
      "carrier": "Progressive",
      "premium": 2500.00,
      "commission_amount": 312.50
    }
  ],
  "pagination": {
    "page": 1,
    "per_page": 20,
    "total": 45
  }
}

Create Policy

POST

Create a new policy record.

POST /policies

Request Body

{
  "policy_number": "AUTO-789012",
  "customer": "XYZ Corp",
  "carrier": "State Farm",
  "policy_type": "AUTO",
  "premium": 3200.00,
  "effective_date": "2024-01-01",
  "commission_rate": 12.5
}

Response

{
  "success": true,
  "data": {
    "id": "pol-5678",
    "policy_number": "AUTO-789012",
    "commission_amount": 400.00,
    "created_at": "2024-01-15T10:30:00Z"
  }
}

Webhooks

Receive real-time notifications when events occur in your account.

Available Events

Policy Events

  • policy.created
  • policy.updated
  • policy.renewed
  • policy.cancelled

Commission Events

  • commission.calculated
  • commission.paid
  • commission.reconciled
  • commission.discrepancy

Webhook Payload

All webhooks send a POST request with this structure:

{
  "id": "evt_1234567890",
  "type": "policy.created",
  "created": "2024-01-15T10:30:00Z",
  "data": {
    "object": {
      "id": "pol-5678",
      "policy_number": "AUTO-123456",
      // ... full object data
    }
  },
  "signature": "sha256=abc123..."  // HMAC signature for verification
}

SDKs & Libraries

Official SDKs make it easy to integrate with the Commission IQ API.

Python SDK

pip install commission-iq
import commission_iq

client = commission_iq.Client(api_key="YOUR_KEY")

# List policies
policies = client.policies.list(status="active")

# Create commission calculation
result = client.commissions.calculate(
    premium=2500.00,
    rate=12.5,
    type="NEW"
)

JavaScript SDK

npm install @commission-iq/sdk
import CommissionIQ from '@commission-iq/sdk';

const client = new CommissionIQ({
  apiKey: 'YOUR_KEY'
});

// List policies
const policies = await client.policies.list({
  status: 'active'
});

// Calculate commission
const result = await client.commissions.calculate({
  premium: 2500.00,
  rate: 12.5,
  type: 'NEW'
});
